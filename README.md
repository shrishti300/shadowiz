trying to make a new node package

# installation

`npm i shadowiz --save`

then...
...
import {shadowiz} from 'shadowiz';
shadowiz({
shadow_type:'soft',
padding: false
});
...

## options

shadowiz supports 2 options, both of which are optional:

- _shadow_type_ - _hard / soft_ (Defaults to soft)
- _padding_- _boolean_ (Defaults to false)
